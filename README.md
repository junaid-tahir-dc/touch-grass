## Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:8080`

## How It Works

1. **Onboarding**: Set up your profile and interests
2. **Daily Challenge**: Get a personalized challenge each day
3. **Challenge Flow**: 
   - View details and preparation steps
   - Start with optional timer and checklist
   - Submit proof (photo + description)
   - Reflect with guided questions
4. **Community**: Share experiences and get inspired by others
5. **Progress**: Track your streak, XP, and badges

## Editing Content

All app content lives in editable seed files:

- `src/data/seed.ts` - Challenges, series, user data
- `src/data/seed.feed.ts` - Community posts  
- `src/data/seed.resources.ts` - Library articles/videos

Simply edit these files and refresh to see changes immediately.
